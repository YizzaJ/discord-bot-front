Bienvenido a la sección de ayuda

En este apartado obtendrás toda la información necesaria para poder añadir y modificar nuevos proveedores de noticias.

Información

Para añdir y modificar proveedores es necesario un conocimiento muy básico de HTML. Simplemente, se debe saber:

  • Abrir HTML de una página: simplemente debes hacer click derecho en el elemento que quieres extraer y seleccionar "inspeccionar elemento."
  • Tag: nombre del elemento HTML.

    • En este caso, "article" es el tag de ese elemento HTML.

  • Class: clase del elemento HTML.

    • La clase comienza con el nombre class (verde) y el contenido es lo que se encuentra en el recuadro rojo.

  • Atributo: otra información que contiene el elemento HTML.

    Nota: los "cmp-ltrk...." no funcionan como atributos.

    • En este caso, "ue-article-id" es un atributo de ese elemento HTML y el contenido es lo que se encuentra en el recuadro rojo.

Se te dará la opción de escoger entre Tag, Class o Atributo para añadir distintos elementos del proveedor. Puedes escoger cualquiera de los tres. Luego, la aplicación realizará una comprobación para evaluar si se obtiene algún elemento.

También se pueden dejar vacios (dejando la opción "Seleccionar") y la aplicacin intentará obtener los elementos de otra manera, pero esto no se seguro que funcione correctamente ya que el codigo HTML de cada página varía mucho con respecto a otras.

Bienvenido a la seccion de ayuda

En este apartado obtendras toda la informacion necesaria para poder anadir y modificar nuevos proveedores de noticias.

Informacion

Para anadir y modificar proveedores es necesario un conocimiento muy basico de HTML. Simplemente, se debe saber:

  • Abrir HTML de una pagina: simplemente debes hacer click derecho en el elemento que quieres extraer y seleccionar "inspeccionar elemento."
  • Tag: nombre del elemento HTML.
  • Class: clase del elemento HTML.
  • Atributo: otra informacion que contiene el elemento HTML.

    Nota: los no funcionan como atributos.

Se te dara la opcion de escoger entre Tag, Class o Atributo para anadir distintos elementos del proveedor. Puedes escoger cualquiera de los tres, luego la aplicacion realizara una comprobacion para evaluar si se obtiene algun elemento.

Tambien se pueden dejar vacios (dejando la opcion "Seleccionar") y la aplicacion intentara obtener los elementos de otra manera, pero esto no se seguro que funcione correctamente ya que el codigo HTML de cada pagina varia mucho con respecto a otras.
